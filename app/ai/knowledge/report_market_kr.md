# 대한민국 시장 분석 기준

리포트는 대한민국 시장을 기준으로 작성한다. 시장 분석에는 고객 수요, 진입 난이도, 성장 가능성을 함께 포함한다. 고객 수요는 인터뷰 답변에서 드러난 불편, 검색 행동, 구매 기준, 예산 제약을 우선 근거로 판단한다. 진입 난이도는 이미 고객 접점이 형성된 플랫폼, 후기 확보 난이도, 지역 경쟁 밀도, 운영 인력 부담을 함께 고려한다. 성장 가능성은 반복 구매, 재방문, 추천, 리뷰 축적, 지역 확장 가능성을 기준으로 본다.

# 초기 창업자와 소규모 브랜드의 저예산 마케팅 원칙

초기 창업자와 소규모 브랜드는 큰 광고비보다 검증 가능한 작은 실험을 우선한다. 월 예산이 적을 때는 네이버 플레이스 기본 정보 정비, 블로그 후기 확보, 인스타그램 릴스 제작, 카카오톡 채널 재방문 안내, 당근마켓 지역 노출처럼 직접 실행 가능한 활동을 선택한다. 모든 전략은 목표 고객, 실행 액션, 사용할 채널, 측정 KPI, 리스크를 함께 제시한다.

# 국내 플랫폼별 활용 기준

네이버 블로그는 검색 기반 관심 고객에게 상세한 경험, 후기, 가격, 위치, 이용 방법을 설명할 때 적합하다. 저예산으로는 고객 후기, 비교 글, 문제 해결형 콘텐츠를 꾸준히 작성한다.

네이버 플레이스는 지역 기반 방문, 예약, 전화 문의, 길찾기 전환에 적합하다. 영업시간, 위치, 사진, 가격, 리뷰 답변을 정비하고 실제 고객 리뷰를 꾸준히 확보한다.

네이버 스마트스토어는 실물 상품, 예약권, 키트, 쿠폰처럼 온라인 결제와 상품 비교가 중요한 경우에 적합하다. 초기에는 대표 상품 수를 줄이고 상세페이지와 리뷰 확보에 집중한다.

카카오톡 채널은 문의 응대, 예약 안내, 재방문 쿠폰, 단골 고객 관리에 적합하다. 과도한 메시지는 차단을 유발할 수 있으므로 고객에게 명확한 혜택이 있을 때만 발송한다.

인스타그램 릴스는 시각적 매력, 제작 과정, 전후 비교, 공간 분위기, 고객 반응을 빠르게 보여줄 때 적합하다. 저예산으로는 스마트폰 촬영과 짧은 편집으로 반복 게시한다.

유튜브 쇼츠는 검색과 추천을 동시에 노릴 수 있는 짧은 교육형, 후기형, 과정형 콘텐츠에 적합하다. 긴 설명보다 한 가지 장면이나 문제 해결을 명확히 보여준다.

당근마켓은 동네 기반 고객, 지역 체험, 근거리 방문, 생활 밀착형 서비스에 적합하다. 지역성과 신뢰가 중요하므로 과장된 표현보다 실제 위치, 가격, 혜택, 이용 장면을 구체적으로 제시한다.

# 경쟁사 분석 기준

직접 경쟁사는 같은 고객 문제를 같은 방식으로 해결하는 국내 서비스, 매장, 플랫폼이다. 간접 경쟁사는 고객의 시간과 예산을 대체로 가져가는 채널, 커뮤니티, 검색 결과, 콘텐츠 채널이다. 대체재는 고객이 굳이 해당 서비스를 쓰지 않고도 문제를 해결하는 방법이다. 확실하지 않은 기업명은 단정하지 말고 "유사 서비스/대체재" 또는 "간접 경쟁 채널"로 표현한다.

# 1개월, 2개월, 3개월 실행 로드맵 기준

1개월 차는 노출과 기본 전환 기반을 만든다. 네이버 플레이스 정비, 핵심 키워드 블로그 글, 릴스 테스트, 기존 고객 후기 요청처럼 바로 실행 가능한 활동을 진행한다.

2개월 차는 반응이 있었던 채널을 반복하고 전환을 높인다. 후기 콘텐츠를 늘리고, 예약 또는 문의 동선을 줄이며, 카카오톡 채널이나 네이버 예약처럼 고객 행동을 이어주는 장치를 실험한다.

3개월 차는 재방문과 추천을 만든다. 재방문 쿠폰, 친구 추천 혜택, 단골 고객 메시지, 지역 제휴, 베스트 후기 재활용을 통해 신규 유입 의존도를 낮춘다.

# 사실 단정 방지 원칙

실제 회사명, 플랫폼명, 경쟁사명은 국내에서 널리 알려진 범위에서만 사용한다. 인터뷰 정보와 지식 문서만으로 존재가 불확실한 기업명은 만들지 않는다. 확신이 낮으면 "유사 서비스/대체재", "간접 경쟁 채널", "지역 내 유사 업체"처럼 범주형 표현을 사용한다.
